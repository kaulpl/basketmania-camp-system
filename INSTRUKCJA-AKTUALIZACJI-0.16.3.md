# Basketmania Camp System 0.16.3

Wersja poprawkowa Panelu Rodzica.

## Najważniejsze poprawki

- usunięto dodatkowe kolorowe koło pojawiające się pod znacznikiem etapu;
- opis etapu nie wychodzi już poza obramowanie kafelka;
- poprawiono zawijanie tekstu na komputerach i telefonach;
- zachowano pięcioetapową oś: Rejestracja, Formularz, Umowa, Płatność, Faktura.

## Aktualizacja

1. Wykonaj kopię zapasową plików i bazy danych.
2. W panelu WordPress przejdź do Wtyczki → Dodaj nową → Wyślij wtyczkę na serwer.
3. Wskaż plik `basketmania-camp-system-0.16.3.zip`.
4. Zastąp obecną wersję wtyczki.
5. Wyczyść pamięć podręczną strony i przeglądarki.
